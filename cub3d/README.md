# cub3d

cub3d is a graphical project from the 42 School common core curriculum, inspired by early raycasting games such as Wolfenstein 3D.

The goal of the project is to render a 3D view from a 2D map using raycasting, while handling player movement, wall collisions, textures and basic scene parsing.

## Features

- Raycasting-based 3D rendering
- 2D map parsing and validation
- Player movement and rotation
- Wall collision handling
- Texture loading and rendering
- Floor and ceiling color management
- Keyboard event handling with MiniLibX

## Concepts

- C programming
- Raycasting
- 2D/3D rendering logic
- Map parsing
- Event handling
- Memory management
- Basic game engine structure

## Usage

Compile the project:

```bash
make
```

Run the program with a map:

```bash
./cub3D maps/map.cub
```

Clean generated files:

```bash
make fclean
```
