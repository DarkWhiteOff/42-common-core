# 42 Common Core

This repository contains a selection of projects completed during the 42 School common core curriculum.

The common core focuses on low-level programming, algorithms, Unix systems, networking, concurrency, object-oriented programming and software engineering fundamentals. Most projects are written in C, with later modules introducing C++, graphics programming and network programming.

## Projects

### Libft

Libft is a custom implementation of common C standard library functions.  
The project focuses on memory management, string manipulation, linked lists, pointers and building reusable low-level utilities from scratch.

### ft_printf

ft_printf is a recreation of the standard `printf` function.  
The project introduces variadic functions, formatted output, format parsing and handling multiple data types in C.

### get_next_line

get_next_line is a function that reads from a file descriptor line by line.  
It focuses on static variables, buffer management, file descriptors and careful memory handling.

### Born2beroot

Born2beroot is a system administration project based on setting up and securing a Linux virtual machine.  
It covers users, groups, SSH, firewall configuration, password policies, sudo rules, partitioning and basic monitoring.

### minitalk

minitalk is a small client-server communication program using Unix signals.  
The project focuses on inter-process communication, signal handling and binary data transmission between processes.

### so_long

so_long is a small 2D game built with the MiniLibX graphics library.  
It introduces map parsing, sprites, keyboard events, rendering, basic game loop logic and path validation.

### Philosophers

Philosophers is a concurrency project based on the dining philosophers problem.  
It focuses on threads, mutexes, race conditions, deadlock prevention and synchronization.

### NetPractice

NetPractice is a networking project focused on TCP/IP addressing and subnetting.  
It covers IP ranges, subnet masks, routing, gateways and basic network troubleshooting.

### C++ Modules

The C++ modules introduce object-oriented programming through a progressive set of exercises.  
They cover classes, inheritance, polymorphism, operator overloading, templates, exceptions, casts and standard containers.

### cub3d

cub3d is a graphical project inspired by early raycasting games such as Wolfenstein 3D.  
It focuses on rendering a 3D view from a 2D map, using raycasting, textures, player movement, wall collisions and map parsing.

### ft_irc

ft_irc is a C++ networking project based on building an IRC server compatible with standard IRC clients.  
It focuses on TCP sockets, multiple client handling, command parsing, channels, private messages, user authentication and protocol implementation.
